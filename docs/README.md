# Centro Operativo Dorado v3

Plataforma modular expandible para:
- gestionar proyectos
- consolidar ideas
- controlar avances por bitácora
- reutilizar módulos
- publicar resultados tipo store
- sincronizar con Supabase
- integrar backend en DigitalOcean
- documentar y versionar en GitHub

## Estructura
- frontend: index.html + css + js
- backend: server.js + routes + services + adapters + models + utils
- docs: arquitectura, etapas, changelog

## Infraestructura objetivo
- GitHub: versionado, Pages, documentación
- Supabase: datos operativos y sincronización
- DigitalOcean: backend, APIs, procesos de consolidación
